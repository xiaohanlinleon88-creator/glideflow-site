Glideflow gives independent developers one OpenAI-compatible endpoint for DeepSeek and Qwen models. You keep the OpenAI SDK shape and point it at the Glideflow base URL.

## 1. Get an API key

Create or log in to your Glideflow dashboard at <a href={siteConfig.appUrl}>{siteConfig.appUrl}</a>. From there you can register, create API keys, and manage credits.

Keys use the familiar `sk-...` bearer token shape.

## 2. Set the base URL

Use <code>{siteConfig.apiBaseUrl}</code> as your OpenAI SDK base URL.
